# Recomendaciones para Imágenes de Productos

## Especificaciones Técnicas

### Formato de Archivo
- **Formatos soportados**: PNG, JPG, JPEG, GIF, WEBP
- **Formato recomendado**: **WEBP** (mejor compresión y calidad)
- **Alternativa**: JPG con alta calidad (85-90%)

### Tamaño de Archivo
- **Máximo**: 16 MB
- **Recomendado**: 200-500 KB
- **Óptimo**: < 300 KB

### Dimensiones de Imagen

#### Catálogo (ProductCard)
- **Recomendado**: 800x800 px (cuadrado 1:1)
- **Mínimo**: 600x600 px
- **Máximo**: 1200x1200 px
- **Aspecto**: Las imágenes se mostrarán completas con padding (no se recortan)

#### Galería de Producto (Modal)
- **Recomendado**: 1200x900 px o 1600x1200 px
- **Aspecto**: 4:3 o 16:9 funciona bien

#### Admin Panel (Thumbnails)
- Las imágenes se redimensionan automáticamente
- Usa las mismas especificaciones del catálogo

---

## Guía de Composición

### 1. Fondo
- **Ideal**: Fondo blanco limpio o gris muy claro (#F5F5F5)
- **Alternativa**: Fondo transparente (PNG/WEBP)
- Evita fondos con texturas o patrones complejos

### 2. Producto en la Imagen
- **Centrado**: El producto debe estar centrado en la imagen
- **Espaciado**: Deja ~10-15% de margen alrededor del producto
- **Ángulo**: Vista frontal o 3/4 para mostrar profundidad

### 3. Iluminación
- Iluminación uniforme y suave
- Evita sombras duras o reflejos excesivos
- Usa luz natural o softboxes

### 4. Enfoque
- Producto completamente enfocado
- Alta nitidez en bordes y detalles
- Sin desenfoque o motion blur

## Proceso de Optimización

### Herramientas Recomendadas

#### Online (Gratuitas)
1. **TinyPNG** (https://tinypng.com)
   - Excelente para JPG y PNG
   - Reduce hasta 70% sin pérdida visible

2. **Squoosh** (https://squoosh.app)
   - Conversión a WEBP
   - Comparación visual lado a lado

3. **Compressor.io** (https://compressor.io)
   - Optimización inteligente
   - Soporta múltiples formatos

#### Software Profesional
- **Adobe Photoshop**: Exportar como "Save for Web"
- **GIMP** (gratuito): Export con calidad 85-90%
- **XnConvert** (gratuito): Batch processing

---

## Checklist Pre-Upload

Antes de subir una imagen, verifica:

- [ ] Formato: WEBP o JPG de alta calidad
- [ ] Tamaño: < 500 KB
- [ ] Dimensiones: 800x800 px (mínimo 600x600 px)
- [ ] Fondo: Blanco/gris claro o transparente
- [ ] Producto: Centrado con margen de ~10%
- [ ] Enfoque: Nítido y bien iluminado
- [ ] Colores: Representan fielmente el producto
- [ ] Sin marcas de agua o texto superpuesto

---

## Múltiples Imágenes por Producto

### Orden Recomendado
1. **Principal**: Vista frontal del producto
2. **Segunda**: Vista lateral o 3/4
3. **Tercera**: Detalle de materiales/acabados
4. **Cuarta**: Producto en contexto (ambiente)
5. **Quinta+**: Detalles adicionales, dimensiones, etc.

### Cantidad Recomendada
- **Mínimo**: 1 imagen (principal)
- **Recomendado**: 3-5 imágenes
- **Óptimo**: 5-8 imágenes variadas

---

## Errores Comunes a Evitar

❌ **NO hagas esto**:
- Imágenes pixeladas o de baja resolución
- Fondos desordenados o distractores
- Iluminación deficiente (muy oscura/clara)
- Producto cortado en los bordes
- Archivos > 2 MB sin optimizar
- Marcas de agua visibles
- Logos de competencia

✅ **SÍ haz esto**:
- Imágenes nítidas y bien iluminadas
- Fondo limpio y profesional
- Producto centrado con espacio
- Múltiples ángulos del producto
- Archivos optimizados < 500 KB
- Colores fieles al producto real
- Consistencia visual entre productos

## Ejemplo de Configuración en Photoshop

```
1. Crear nuevo documento: 800x800 px, 72 DPI
2. Fondo: #FFFFFF (blanco)
3. Importar foto del producto
4. Ajustar: Brillo/Contraste, Saturación
5. Recortar/Redimensionar para centrar
6. Exportar:
   - Archivo > Exportar > Save for Web (Legacy)
   - Formato: JPEG
   - Calidad: 85-90
   - Optimizado: ✓
   - Tamaño objetivo: < 300 KB
```

## Soporte Técnico

Si tienes problemas con la subida de imágenes:

1. Verifica que el archivo sea < 16 MB
2. Asegúrate de usar formato soportado (PNG/JPG/WEBP)
3. Intenta optimizar con TinyPNG o Squoosh
4. Si el error persiste, contacta al equipo técnico
